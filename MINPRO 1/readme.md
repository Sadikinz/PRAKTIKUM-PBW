## Penjelasan Setiap Section 

### Section Navbar

Pada bagian ini digunakan untuk melakukan navigasi ke section lainnya 

Fitur:
- Terdapat menu sekaligus tombol untuk melakukan navigasi ke Home, About, dan Certificates 
- akan selalu berada diatas karena menggunakan fixed-top
- ada efek scroll saat menu di klik

### Hero Section

Tampilan Pembuka dari Website seperti nama, sebagai apa, dan qoutes, semua teks diambil dari vue.js

Fitur:
- Teks Bersifat dinamis karena menggunakan vue.js
- Background terdapat efek glow warna biru

### Section About Me

Pada bagian ini terdiri dari foto, deskripsi singkat, dan skill bar 

Fitur:
- foto profil berbentuk lingkaran
- Menggunakan sistem grid Bootstrap
- Semua kata dan persentase di skill bar berasal dari Vue.js

### Section Certificates dan Footer

Pada bagian ini menampilkan Sertifikat Pengalaman dalam bentuk Card/Kartu dan footer

Fitur:
- Menggunakan sistem looping dari Vue.js 
- Data Sertifikat disimpan dalam array
- setiap kartu memiliki gambar dan deskripsi

Fitur Footer:
- Menampilkan copyright
- Menampilkan ikon media sosial
- Menggunakan ikon dari Bootstrap
